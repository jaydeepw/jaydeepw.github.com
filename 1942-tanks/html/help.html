Help




What if the Game doest run at all

JavaScript might have been disabled. Goto wrench icon in the top right corner, click Options and Enable JavaScript.

What if the Game Runs Slow

Sorry to say, but either you have a pretty old computer or the Google Chrome Browser version is old. Click the wrench icon in the top right corner, then goto Tool->About Google Chrome. It will soon start upgrading to latest version

What if the Game freezes

The Google Chrome Browser version is pretty old. Click the wrench icon in the top right corner, then goto Tool->About Google Chrome. It will soon upgrading to latest version

Game Sound is not audible

Check if your system has sound enabled and appropriate drivers are been installed on the system
